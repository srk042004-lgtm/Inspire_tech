Class 11: If-Else Conditions (Decision Making)

In programming, decision-making is handled by conditional statements, primarily the if, elif, and else keywords. These allow a program to execute specific blocks of code only when certain logical conditions are met, such as checking if a user is logged in or if a sensor value exceeds a limit. Python uses indentation to define which lines of code belong to which condition, making the logic visually clear and easy to follow. You can use comparison operators like == or != to build these rules, and combine them with logical operators like and or or for more complex requirements. For example, in an AI system, an if statement might decide whether to trigger an alarm based on a probability score from a vision model. Mastering conditions is the first step toward creating software that feels "intelligent" rather than just robotic. Without them, programs would simply run in a straight line without reacting to different inputs or environmental changes. 

if marks >= 40:
    print("You passed!")
else:
    print("Try again.")

Class 12: The 'For' Loop (Iteration)

The for loop is used to iterate over a sequence, such as a list, a string, or a range of numbers, executing a block of code for each item in that sequence. This is incredibly powerful for tasks like processing thousands of images in a dataset or sending a personalized email to every student in a database. Instead of writing the same line of code fifty times, a for loop allows you to write it once and tell Python how many times to repeat it. We often use the range() function with for loops to generate a sequence of numbers on the fly, such as range(1, 10). In Artificial Intelligence, for loops are the primary tool used to "train" models by passing data through a neural network multiple times to improve accuracy. They simplify your code, reduce errors, and make your programs much more scalable for large-scale data processing. Understanding loops is essential for any developer looking to automate repetitive manual work in the modern digital age. 

for i in range(5):
    print("Inspire Tech Learning...")

Class 13: The 'While' Loop

Unlike the for loop which runs for a specific number of times, a while loop continues to execute as long as a certain condition remains True. This makes it ideal for situations where you don't know exactly how many repetitions are needed, such as waiting for a user to enter the correct password or keeping a game running until the player quits. A while loop checks its condition before every "lap," and if the condition ever becomes False, the loop stops immediately. It is crucial to include a way to change the condition inside the loop (like an incrementing counter), otherwise, you might create an "Infinite Loop" that crashes your program. In AI robotics, while loops are often used to keep a robot's sensors active and scanning the environment until it reaches its target destination. They provide a level of dynamic control that is essential for building long-running applications and interactive user interfaces. Mastering the while loop ensures your programs can handle unpredictable durations and continuous real-time processes effectively. 

count = 1
while count <= 5:
    print("Iteration:", count)
    count += 1
