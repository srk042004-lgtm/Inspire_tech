Class 13: Ek Mukammal Webpage Banana (Creating a Webpage using HTML)

Is class mein hum ab tak seekhe hue sabhi HTML concepts—jaise structure, headings, links, images, tables, aur forms—ko mila kar ek chota (mini) lekin mukammal webpage banayenge. Yeh class aapki saari skills ko consolidate (mazboot) karne ka kaam karegi.

Video Lesson: Page Layout Aur Integration

1. Webpage Ke Teen (3) Essential Parts

Har achhe webpage ko hum teen bade hisson mein baant sakte hain, jinhe hum semantic tags se define karte hain:

A. Header (`<header>`)

Ise website ka maatha (head) samjhiye. Isme logo, website ka naam, aur main navigation links (`<nav>`) aate hain.

B. Main Content (`<main>`)

Yeh woh hissa hai jahan webpage ka asali (primary) content aata hai. Isme articles, sections, tables, aur forms jaise elements shaamil hote hain.

C. Footer (`<footer>`)

Yeh webpage ka sabse nichla (bottom) hissa hota hai. Isme aam taur par copyright, contact details, aur secondary links diye jaate hain.

2. Demo HTML Webpage (HTML Sirf)

Yahan ek chota sa HTML code diya gaya hai jo ek company ke "About Us" page ko darshata hai. Isme image, table aur form ka istemaal kiya gaya hai: 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Hamari Company Ke Baare Mein</title>
</head>
<body>

    <header>
        <h1>Future Tech Solutions</h1>
        <nav>
            <a href="#about">About Us</a> |
            <a href="#team">Hamari Team</a> |
            <a href="#contact">Contact</a>
        </nav>
        <hr>
    </header>

    <main>
        <section id="about">
            <h2>Hamara Mission</h2>
            <img src="https://placehold.co/400x150/007bff/ffffff?text=Company+Logo" alt="Tech Company Logo" width="400">
            <p>Hum 2020 se market mein hain aur naye zamane ki technology se duniya ko badalna chahte hain. Humari team...</p>
            <p>Aur zyada jaankari ke liye, <a href="details.html">yahan click karein</a>.</p>
        </section>

        <section id="team">
            <h2>Hamari Core Team</h2>
            <table border="1" style="width: 80%;">
                <tr>
                    <th>Naam</th>
                    <th>Designation</th>
                    <th>Experience</th>
                </tr>
                <tr>
                    <td>Ahmed Khan</td>
                    <td>CEO</td>
                    <td>15 Years</td>
                </tr>
                <tr>
                    <td>Fatima Ali</td>
                    <td>CTO</td>
                    <td>10 Years</td>
                </tr>
            </table>
        </section>

        <section id="contact">
            <h2>Humse Raabta Karein (Contact Us)</h2>
            <form action="/submit-form" method="post">
                <label for="name">Aapka Naam:</label>
                <input type="text" id="name" name="user_name" required><br><br>

                <label for="email">Email:</label>
                <input type="email" id="email" name="user_email" required><br><br>

                <input type="submit" value="Message Bhejein">
            </form>
        </section>
    </main>

    <footer>
        <hr>
        <p>&copy; 2025 Future Tech Solutions. All rights reserved.</p>
    </footer>

</body>
</html>
