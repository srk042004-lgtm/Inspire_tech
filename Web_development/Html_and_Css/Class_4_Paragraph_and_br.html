📖 HTML Paragraph, Line Break & Horizontal Rule

Learn how to structure text content using simple but powerful tags.

📝 1️⃣ Paragraph Tag <p>

The <p> tag defines a block of text known as a paragraph. Browsers automatically add space before and after it. 

<p>This is my first paragraph.</p>
<p>This is another paragraph.</p>

This is my first paragraph.

This is another paragraph.

🔹 2️⃣ Line Break Tag <br>

The <br> tag is used to break a line inside a paragraph or text block. It is an empty tag (no closing tag). 

<p>This is a line.<br>This is another line after break.</p>

This is a line.
This is another line after break.

➖ 3️⃣ Horizontal Rule Tag <hr>

The <hr> tag inserts a horizontal line that divides content sections. It’s often used for visual separation. 

<h2>About Us</h2>
<p>We are learning HTML.</p>
<hr>
<h2>Contact Us</h2>
<p>Email: info@example.com</p>

About Us

We are learning HTML.

Contact Us

Email: info@example.com

🧠 Practice Task: Try It Yourself

Create a webpage using all the tags you learned:

  • Use at least 2 paragraphs.
  • Add a line break with <br>.
  • Separate sections with a horizontal rule <hr>. 
<h1>My Webpage</h1>
<p>Welcome to my HTML lesson.</p>
<hr>
<p>I love coding.<br>I also enjoy teaching.</p>
